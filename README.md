# quickapp-neihanchezhan
基于快应用开发的内涵车站APP

# 安装
```
git clone git@github.com:Licoy/quickapp-neihanchezhan.git

npm install

npm run watch / npm run build / npm run server
```
PS：nodejs版本必须为6.x版本，更高版本则不兼容。
